# CaterTrack Landing Page

Landing page corporativa para CaterTrack, desarrollada con HTML5 semántico, CSS modular, Bootstrap 5, AOS y JavaScript ES6.

## Estructura

- index.html: página principal.
- css/: estilos base, variables y responsive.
- js/: interacciones, animaciones, formulario y WhatsApp.
- assets/: imágenes, iconos, logos y fuentes.

## Vista previa

Abrir index.html en un navegador o servir la carpeta con un servidor estático.
